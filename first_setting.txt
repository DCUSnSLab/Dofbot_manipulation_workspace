IP 확인 명령어 : $ifconfig
IP 확인 후, 하기된 변수를 .bashrc에 export 추가할 것.
$ROS_MASTER_URI=http://{HOST PC의 IP}:11311
$ROS_HOSTNAME={해당 PC의 IP}

패키지 빌드 및 스크립트 실행 셋업
$ cmd
$ source /opt/ros/noetic/setup.bash
$ source ~/.bashrc
$ cd ~/{WORKSPACE}
$ catkin_make
$ source devel/setup.bash
$ roslaunch ...
